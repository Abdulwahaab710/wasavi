wasavi options

exrc

The contents of this text box are the initialization commands for wasavi. There commands will be executed for every starting of wasavi editor. You can write any ex commands you like.

target elements

shortcut key

  • Shortcut format is <(modifier-)*key>,...
    For example, <c-enter>, <insert> means "If ctrl+enter or insert is pressed, then invoke wasavi."
  • Following keys can be used as shortcut: backspace, bs, tab, enter, return, ret, pageup, pagedown, end, home, left, up, right, down, insert, ins, delete, del, f1 to f12, a to z, and 0 to 9.
  • Following keys can be used as modifier: s (means SHIFT), c (means CTRL).
  • Leaving the text field empty, default shortcut <c-enter>, <insert> will be used.
